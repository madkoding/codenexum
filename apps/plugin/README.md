# @codenexum/plugin — opencode CodeNexum plugin

Thin MCP client that connects to the CodeNexum Electron app and registers `context_*` tools in opencode.

## Install

Add to your `opencode.json`:

```json
{
  "plugins": ["@codenexum/plugin"]
}
```

Restart opencode. The plugin auto-discovers the Electron app.

## Configuration

- `CODENEXUM_MCP_URL` — override MCP endpoint (optional)
- `~/.config/codenexum/mcp.json` — auto-generated by the Electron app

See [CONFIG.md](../../CONFIG.md) for all options.

## How it works

1. Discovers the MCP endpoint (env var → mcp.json → default `http://127.0.0.1:7770`)
2. Opens an MCP client connection via `@opencode-ai/plugin`
3. Registers 7 `context_*` tools that forward to MCP tools
4. Intercepts `read`/`bash`/`grep`/`glob` outputs for compression
5. Flushes per-turn savings to the MCP server on `session.idle`

The plugin has no indexer, no SQLite, no compression logic — all of that lives in the Electron app.
